# Task Manager с маршрутизацией

Веб-приложение для управления задачами с системой маршрутизации и статистикой.

## Описание

Task Manager — SPA-приложение для управления задачами, разработанное на React. Проект демонстрирует работу с маршрутизацией, формами, валидацией и управлением состоянием.

## Функциональность

### Основные возможности

- Создание, редактирование и удаление задач
- Фильтрация задач по статусу (все/активные/выполненные)
- Сортировка по дате и приоритету
- Детальная страница для каждой задачи
- Статистика выполнения задач
- Адаптивный дизайн

### Страницы приложения

- `/` — главная страница со списком задач
- `/create` — создание новой задачи
- `/task/:id` — детальная информация о задаче
- `/stats` — статистика по задачам
- `*` — 404 страница

## Технологии

- **React 18** — библиотека для построения UI
- **React Router v6** — маршрутизация
- **React Hook Form** — управление формами и валидация
- **Tailwind CSS** — стилизация
- **Vite** — сборщик проекта
- **LocalStorage** — хранение данных

## Установка и запуск

```bash
# Клонирование репозитория
git clone <repository-url>
cd task-manager

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для production
npm run build
```

После запуска приложение доступно по адресу `http://localhost:5173`

## Что реализовано

### Главная страница

- Список всех задач с возможностью быстрого изменения статуса
- Фильтрация: все, активные, выполненные
- Сортировка: по дате создания, по приоритету
- Удаление задач

### Создание задачи

- Форма с полями: название, описание, приоритет, дедлайн
- Валидация полей в реальном времени
- Автоматический переход на главную после создания

### Детальная страница задачи

- Полная информация о задаче
- Редактирование данных
- История изменений
- Навигация обратно к списку

### Статистика

- Общее количество задач
- Количество выполненных и активных задач
- Статистика по приоритетам
